# MY-PORTFOLIO

業務改善ツール、自動化スクリプト、Bot、シミュレーターなど、  
**実務課題・個人課題を解決するために開発したプロジェクトのポートフォリオ**です。

「現場で本当に使えるか」「非エンジニアでも運用できるか」を重視し、  
小規模なツールであっても **設計・安全性・継続運用** まで考えた実装を行っています。

## ChatGPT の活用について

ChatGPT は **設計や判断を任せるための存在ではなく、  
実装作業を効率化するための補助ツール**として活用しています。

業務フローの理解、仕様整理、UI 構成、データ構造などの設計判断は自分で行い、  
その上で **確定した設計をコードに落とす工程**を補助させています。

主な使い方は、

・既に決まっている仕様に基づくコード生成  
・定型処理の記述量削減  
・既存コード構造を前提とした修正案の整理

などです。

生成されたコードは必ず内容を確認し、  
既存仕様・実データ・運用と照合したうえで調整しています。

ChatGPT は「考える代替」ではなく、  
**考えた結果を素早く形にするための道具**という位置付けです。
