# Skills

これまでの業務改善・個人開発を通じて使用してきた技術スタックです。  
すべて「実際に使って動かし、運用まで考えた経験」があるものを記載しています。

---

## プログラミング言語

- **Python**
  - 業務自動化、データ処理、Bot、シミュレーター開発
  - Selenium / OpenCV / pytesseract / Tkinter / Pygame などを利用
  - exe 配布（PyInstaller）を含む運用経験あり

- **JavaScript**
  - Chrome 拡張、Web ツール、Bot フロントエンド
  - DOM 操作、イベント駆動設計、Vanilla JS 中心

- **VBA (Excel)**
  - UserForm を用いた業務ツール開発
  - 長期運用・シート構造変更耐性を意識した設計

## データ・設計

## 開発スタイル・強み

- 非エンジニア利用を前提とした UI・UX 設計
- 実データを用いた検証と改善
- 小規模でも壊れにくい構造設計
- 安全性・説明責任を意識した自動化判断
