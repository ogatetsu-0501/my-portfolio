# 会計システム移行自動化ツール（ジョブカン会計）

旧会計システムからクラウド会計ソフトへの移行時に発生した  
大量の仕訳再登録作業を自動化するために開発したツールです。

---

## 概要

引き継ぎミスにより、  
**3か月分・3万件超の仕訳を手入力で再登録する必要**が発生。

経理担当者の負担軽減と誤入力防止を目的に、  
CSVを解析し、ブラウザ操作で安全に自動入力するツールを  
**3日で開発・即投入**しました。

---

## 主な機能

- CSV読込・エンコード自動判定
- 伝票単位でのグルーピング登録
- 重複防止・差額チェック
- 進捗表示・エラーログ出力

---

## 使用技術

- Python 3
- Selenium
- Tkinter
- CSV / 正規表現
- PyInstaller

---

## 実装上の工夫

- CSVインポートを使わずUI操作を自動化（安全性重視）
- 日本語会計データ特有の揺れを吸収する正規化処理
- 非エンジニアでも使えるexe配布

---

## 成果・効果

- 手入力作業を実質ゼロに削減
- 誤登録なしで移行完了
- スピード感と実務対応力を評価
