# Discord 連帯責任リマインダー Bot

通知を無視しがちな人でも行動できるよう、  
**軽い社会的プレッシャーを仕組みに組み込んだリマインダーBot**です。

---

## 概要

ADHD傾向のある友人の課題をきっかけに開発。  
スヌーズし続けると、サーバー内のランダムメンバーに  
メンションが飛ぶ仕組みを実装しています。

---

## 主な機能

- 定期リマインド通知
- スヌーズ回数に応じたランダムメンション
- タイムゾーン・quiet hours 対応
- 多言語対応

---

## 使用技術

- Python
- Firestore
- Google Cloud
- Discord API

---

## 実装上の工夫

- 無料枠で永続稼働するイベント駆動構成
- 権限制御（保証人のみタスク完了報告可能）
- 拡張を見据えたデータ設計

---

## 成果・効果

- 行動促進に実際に効果あり
- 個人・小規模サーバーで運用可能
- UXとロジックを両立したBot設計
