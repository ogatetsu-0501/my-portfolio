# ゲーム育成シミュレーター（静的攻略ツール）

ゲームの育成要素を分析するための  
**完全クライアントサイド型 Web シミュレーター**です。

---

## 概要

序盤行動が終盤結果に影響する育成要素を、  
1〜60ターン全てシミュレーション可能な形で可視化しました。

---

## 主な機能

- 全ターン時系列シミュレーション
- 確率分布による期待値計算
- JSONインポート／エクスポート
- オフライン即起動

---

## 使用技術

- HTML / CSS
- Vanilla JavaScript
- localStorage

---

## 実装上の工夫

- 長期状態を一意キーで管理
- DPによる高速確率計算
- ゲームUIに寄せた直感的レイアウト

---

## 成果・効果

- 攻略検証時間を大幅短縮
- アップデート初日から利用可能
- 軽量・高速な静的ツール設計を実践
