# ゲーム内マーケット分析・可視化ツール開発

オンラインゲームの公式アバターオークション情報を収集・分析し、  
価格や性能を比較しやすく可視化するために開発した個人プロジェクトです。

---

## 概要

公式マーケットは検索・比較機能がシンプルで、

- 過去の価格推移が分からない
- 性能と価格の妥当性を比較しづらい
- 所持アイテムとの差分が把握しにくい

といった課題がありました。

本ツールでは、取引データを自動収集・集計し、  
**購入判断を支援する可視化UI** を提供しています。

---

## 主な機能

- マーケット取引データの定期自動取得
- 価格履歴の蓄積・集計
- 独自スコアによるコストパフォーマンス算出
- 条件フィルタ・価格推移グラフ表示

---

## 使用技術

### バックエンド
- Node.js
- Google Cloud Functions
- GitHub REST API
- p-limit（並列数制御）
- node-fetch

### フロントエンド
- HTML / CSS / JavaScript
- Chart.js

---

## 実装上の工夫

- Cloud Functions による定期実行パイプラインを構築し、PC非依存でデータ更新
- GitHub API 経由で JSON データを自動コミット・配信
- p-limit とリトライ処理により、大量データ取得時の安定性を確保
- ステータスを数値化した独自指標を用いて、直感的な比較を可能に
- 更新頻度・通信量・リージョン構成を考慮し、クラウド運用コストを最適化

---

## 成果・効果

- 価格・性能の比較が容易になり、購入判断時間を大幅に短縮
- データ取得から可視化までのフルスタック構成を個人で実装
- クラウド運用とコスト意識を踏まえた設計経験を獲得
