# 勤怠入力補助 Chrome 拡張

社内の勤怠入力作業を支援するために開発した  
**Chrome拡張機能（業務改善ツール）**です。

---

## 概要

勤怠入力画面において、

- 当日の行が分かりにくい
- 毎回同じ内容を入力している
- 入力漏れが発生しやすい

といった課題を解決するため、  
**既存画面を壊さずに操作性を改善する拡張**として設計しました。

---

## 主な機能

- 当日の行を自動検出してスクロール・ハイライト表示
- 入力テンプレートの保存・再利用
- 視認性向上のための動的CSS注入

---

## 使用技術

- JavaScript
- Chrome Extensions API
- chrome.storage.local
- DOM操作

---

## 実装上の工夫

- DOM構造に過度に依存しない要素取得ロジック
- 初回起動時の自動初期化処理
- ユーザー別設定を想定したデータ設計
- content script 単体で完結する軽量構成

---

## 成果・効果

- 部署内で実運用され、入力漏れ防止に貢献
- 勤怠入力時間の短縮
- 「小さくても効く」業務改善事例として評価
