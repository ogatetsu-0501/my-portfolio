# 画像認識＋自動操作ツール開発

画面上のUI要素や表示内容を画像認識・OCRで検出し、  
繰り返し作業や検査・データ確認業務を自動化するために開発したツール群です。

---

## 概要

人手で行っていた以下のような作業を対象にしています。

- 画面上のボタンや状態を目視で確認して操作する
- 表示されている数値や文字列を読み取って判断する
- 同じ操作・確認を何度も繰り返す

これらを **画像認識＋OCR＋自動操作** の組み合わせで置き換え、  
人の判断を極力ロジック化することで作業効率と再現性を高めました。

---

## 主な機能

- テンプレートマッチングによるUI要素検出
- OCRによる画面内テキストの取得
- 認識結果に基づく処理分岐・操作順序の自動制御
- 認識精度検証用のデバッグ支援機能
- 解像度差・表示条件差への対応

---

## 使用技術

- Python
- OpenCV
- pytesseract
- NumPy

---

## 実装上の工夫

- テンプレートマッチングに加え、二値化・しきい値調整を組み合わせて認識精度を安定化
- OCR結果をそのまま使わず、正規化・検証を行ったうえで判定ロジックに連携
- 誤検出時に備え、認識失敗画像を自動保存するデバッグ用スクリプトを作成
- 画面解像度やUI配置差異に対応するため、座標確認・補正用の補助ツールを自作

---

## 成果・効果

- 繰り返し作業や検査・確認業務を自動化
- 人為的な見落としや判断ブレを削減
- 作業時間短縮と再現性向上を実現
